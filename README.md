# Drone-Delivery 
## Unity games project 
### Multiplayer game made using mirror networking. Allows for up to 4 players to join one lobby.
* Controls: WASD - Move Drone, SPACE- Up CTRL - Down, Q and E - Rotate. F- use ability
* Delivery Boxes before timer runs out.
* Each player's score is tracked.
* Different abilities can be picked up and activated
* Dodge obstacles
